# Contributing to SwiftFix 🤝

We're excited that you're interested in contributing to SwiftFix! Whether you're fixing a bug, improving documentation, or suggesting a new feature, your help is appreciated.

## How to Contribute

### 1. Report Bugs
If you find a bug, please open an issue on GitHub and include:
- A clear description of the problem.
- Steps to reproduce the issue.
- Expected vs. actual behavior.

### 2. Suggest Features
We love new ideas! To suggest a feature:
- Open a "Feature Request" issue.
- Describe the feature and why it would be useful.

### 3. Submit Pull Requests
1.  **Fork** the repository.
2.  **Create a branch** for your feature/fix (`git checkout -b feature/amazing-feature`).
3.  **Commit** your changes (`git commit -m 'Add some amazing feature'`).
4.  **Push** to the branch (`git push origin feature/amazing-feature`).
5.  **Open a Pull Request**.

## Code Style
- Use clean, commented code.
- Follow the existing project structure.
- Ensure all TypeScript types are correctly defined.
